# Lesson 8: For Loop — Count and Repeat 🔁

> **Goal:** Use `for` loops with `range()` to repeat actions, count, and accumulate values.

---

## 1. What is a Loop?

Imagine you want to print "Hello" 5 times. Without a loop, you would write `print("Hello")` five times. With a loop, you write it **once** and tell Python to repeat it.

> 💡 **Scratch connection:** The `for` loop in Python is like the **"repeat N"** block in Scratch!

| Scratch | Python |
|---------|--------|
| `repeat 5` block | `for i in range(5):` |
| Code inside the repeat block | Indented code inside the loop |
| The repeat block ends | Loop ends when `range` is exhausted |

---

## 2. Basic `for` Loop

```python
for i in range(5):
    print("Hello")
```

**Output:**
```
Hello
Hello
Hello
Hello
Hello
```

- `for i in range(5):` — repeat 5 times (i = 0, 1, 2, 3, 4).
- The **indented** code below is the **loop body** — it runs each time.
- `i` is the **loop variable** — it counts each iteration.

> ⚠️ **Indentation matters!** The code inside the loop must be indented (4 spaces or 1 Tab).

---

## 3. The Loop Variable `i`

The loop variable `i` changes each time the loop runs:

```python
for i in range(5):
    print(i)
```

**Output:**
```
0
1
2
3
4
```

`range(5)` gives: 0, 1, 2, 3, 4 — it starts at **0** and stops **before** 5.

---

## 4. `range()` Variants

### `range(start, stop)` — choose where to start

```python
for i in range(1, 6):
    print(i)
```

**Output:**
```
1
2
3
4
5
```

`range(1, 6)` gives: 1, 2, 3, 4, 5. The **stop** value (6) is **not included**.

### `range(start, stop, step)` — skip by step

```python
for i in range(1, 10, 2):
    print(i)
```

**Output:**
```
1
3
5
7
9
```

`range(1, 10, 2)` gives every **2nd** number starting from 1.

### `range(start, stop, -1)` — count down

```python
for i in range(5, 0, -1):
    print(i)
```

**Output:**
```
5
4
3
2
1
```

Use a **negative step** to count backwards.

---

## 5. The Accumulator Pattern

An **accumulator** is a variable that collects (accumulates) a result as the loop runs.

```python
total = 0               # start the accumulator at 0
for i in range(1, 6):   # i = 1, 2, 3, 4, 5
    total = total + i   # add i to total each time
print(total)            # print the final result
```

**Output:**
```
15
```

This calculates 1 + 2 + 3 + 4 + 5 = 15.

> 💡 Think of `total` as a running score — you keep adding to it each round.

---

## 6. Printing Patterns with Loops

You can use the loop variable to print patterns:

```python
for i in range(1, 6):
    print("*" * i)
```

**Output:**
```
*
**
***
****
*****
```

`"*" * i` means: repeat the `"*"` character `i` times.

## 7. Vocabulary

| English | Vietnamese | Meaning |
|---------|-----------|---------|
| loop | vòng lặp | A block of code that repeats |
| iteration | lần lặp | One single run through the loop body |
| range | phạm vi | A sequence of numbers generated by `range()` |
| start | bắt đầu | The first number in a range |
| stop | dừng lại | The number where range stops (not included) |
| step | bước nhảy | How much the loop variable increases each time |
| accumulator | bộ tích lũy | A variable that collects a running total |
| counter | bộ đếm | A variable that counts how many times something happens |
| body | thân vòng lặp | The indented code inside the loop |
| indent | thụt lề | Spaces at the start of a line to show it's inside a block |
| repeat | lặp lại | Do something again |
| sequence | dãy số | A list of numbers in order |

---

## 8. Summary

✅ `for i in range(N):` repeats N times (i goes from 0 to N-1).
✅ `range(start, stop)` — start is included, stop is NOT included.
✅ `range(start, stop, step)` — jump by step each time.
✅ Use a negative step to count down.
✅ The accumulator pattern: start at 0, add to it inside the loop.
✅ `"*" * i` repeats a string i times.

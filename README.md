# Python for Kids 🐍

A Python programming course designed for Grade 3 students who already know Scratch.

## Course Overview

- **Target:** Grade 3 students with Scratch experience (variables, loops, lists).
- **Tool:** Thonny IDE, Python 3, Windows.
- **Focus:** Logical thinking, math, turtle graphics.
- **Language:** English materials, bilingual teaching (Vietnamese + English).

## How to Use

1. Open Thonny IDE or Python Lab in Code.org.
2. Follow the textbook for each lesson.
3. Run the example code files.
4. Do the homework exercises (Easy / Medium / Hard).
5. Check your answers in `answers.md`.
